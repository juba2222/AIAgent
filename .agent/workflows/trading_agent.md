# تشغيل وكيل التداول وتحليل سهم معين

هذا المسار (Workflow) يقوم بتشغيل وكيل التداول المبني على مهارات `alphaear`.

1. أولاً، تأكد من أنك قرأت موجه النظام الخاص بالوكيل (Agent System Prompt).
   - اقرأ الموجه من مسار المشروع أو من العقل (brain) لتأطير شخصية الذكاء الاصطناعي الخاص بك.
   
2. قم بتفعيل مهارة استخراج الأخبار `alphaear-news` و `alphaear-sentiment` لتحليل وضع السوق العام وتوجهاته.
   استخدم الأدوات المخصصة للبحث عن الأخبار المتعلقة بالسوق (Macro).

3. استخرج البيانات الفنية للأصل المطلوب باستخدام `alphaear-stock`.

4. حلل الترابط وقم ببناء سلسلة المنطق بين الخبر والسعر عن طريق `alphaear-logic-visualizer`.

5. أخيراً، قم بإنشاء تقرير نهائي باستخدام `alphaear-reporter` يحتوي على:
   - حالة السوق.
   - حالة القطاع.
   - الاستراتيجية المقترحة.
   - نقاط الدخول المتوقعة والمخاطرة.
