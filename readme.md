# 🤖 Alpha Prime: Senior Macro Algo-Strategist

وكيل ذكاء اصطناعي فائق التطور يعالج **14 طبقة** من البيانات الاستخباراتية (من مؤشرات الماكرو إلى التنبؤات العصبية) لإنتاج تقارير تداول مؤسساتية قائمة على **الإطار الديناميكي للارتباطات**، نظرية المزاد (AMT)، وتتبع السيولة الذكية.

---

## 🧠 الهوية والرؤية
**Alpha Prime** ليس مجرد بوت تداول، بل هو **محلل كمي استراتيجي (Senior Macro Algo-Strategist)**. تم تصميمه لفك شفرة تعقيدات الأسواق العالمية وتقديم منطق تداول (Logic) بدلاً من مجرد إشارات (Signals)، مما يجعله الجسر المثالي بين التحليل الاستخباري والتنفيذ الخوارزمي.

> **المبدأ الحاكم:** الأسواق لا تتحرك في فراغ؛ الارتباطات هي "كائنات حية" تتغير مع تغير أنظمة التضخم والسيولة.

---

## 🛡️ الطبقات الاستخباراتية الـ 14 (Intelligence Matrix)
يعمل النظام بعمق غير مسبوق عبر تحليل:
1.  **Price Action:** السلوك السعري والزخم.
2.  **AMT (Auction Theory):** مناطق القيمة والسيولة المفقودة.
3.  **Macro Indicators:** بيانات FRED الحقيقية (CPI, M2, Rate).
4.  **Intermarket Ratios:** النسب القيادية (Copper/Gold, Yield Curve).
5.  **Market Regimes:** كشف النظام (Goldilocks vs Stagflation).
6.  **Smart Money:** تتبع تدفقات الحيتان وأعضاء الكونجرس.
7.  **Sentiment Analysis:** تحليل المشاعر عبر DeepEar & FinBERT.
8.  **Fundamentals:** +150 مؤشر مالي عبر FinanceToolkit.
9.  **Alternative Data:** اللوجستيات وسلاسل التوريد (BDI).
10. **Dynamic Correlations:** اختبارات السببية (Granger Causality).
11. **Volatility & Credit:** مراقبة VIX و MOVE.
12. **Neural Predictions:** التنبؤات السعرية عبر نماذج Kronos.
13. **Signal Evolution:** تتبع قوة ومنطق الإشارة زمنياً.
14. **Risk & Conviction:** حساب درجة اليقين ونقاط الإبطال.

---

## 📊 نظم الأسواق (Market Regimes)
يستخدم Alpha Prime محرك استدلال متقدم لتصنيف السوق وتعديل الاستراتيجية آلياً:
| النظام | الأصول المتفوقة | استراتيجية Alpha Prime |
|:---|:---|:---|
| **النمو المستقر** | أسهم + سندات | تعظيم المخاطر (Risk-On) |
| **التضخم الركودي** | ذهب + سلع + نقد | تحوط دفاعي (Defensive) |
| **الركود الانكماشي** | سندات + ذهب | التركيز على الملاذات |
| **الانتعاش التضخمي** | سلع + أسهم دورية | استغلال الدورة الاقتصادية |

## 🛠️ القدرات التقنية الفائقة
*   **التكامل المؤسساتي:** ربط كامل مع **OpenBB Platform** و **FinanceToolkit**.
*   **محرك AMT المتقدم:** تحليل مستويات السيولة لـ 6 فترات زمنية متداخلة.
*   **الصياغة الخوارزمية:** مخرجات جاهزة للتحويل إلى Pine Script أو MQL5.
*   **نظام التشغيل الموحد:** سير عمل مؤتمت (One-Command Workflow) لإنتاج التقارير.

## 🚀 التشغيل السريع (Quick Start)

### 1. إعداد البيئة
```powershell
# ضبط مفتاح Gemini 
$env:GEMINI_API_KEY="your-key-here"

# تثبيت المتطلبات
pip install -r requirements.txt
```

### 2. تشغيل الوكيل (14 طبقة)
شغّل الدورة الاستخباراتية الكاملة لأي أصل مالي:
```powershell
# مثال لتحليل سهم NVDA
/alpha_prime_agent --asset NVDA --report both
```
