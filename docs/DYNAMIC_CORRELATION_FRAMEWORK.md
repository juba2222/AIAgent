# 🔬 الإطار الديناميكي للارتباطات بين الأصول (Dynamic Correlation Framework)

> **المبدأ الأساسي:** الارتباطات ليست ثوابت رياضية، بل ديناميكيات متغيرة تتأثر بنظام السوق (Market Regime)، ونوع الصدمة، ومستوى التضخم، وحالة السيولة العالمية.

---

## 1. تصنيف نظم الأسواق (Market Regime Classification)

تتشكل نظم الأسواق من التفاعل بين **زخم النمو** و**زخم التضخم**:

| النظام الاقتصادي | نمو GDP | التضخم | الأصول المتفوقة | الأصول المتعثرة | ارتباط أسهم/سندات |
|:---|:---|:---|:---|:---|:---|
| **النمو المستقر (Goldilocks)** | مستقر/صاعد | منخفض/مستقر | الأسهم، السندات | النقد، السلع | سلبي أو منخفض |
| **التضخم الركودي (Stagflation)** | متباطئ/سالب | مرتفع/صاعد | الذهب، السلع، النقد | السندات، الأسهم | **إيجابي مرتفع** ⚠️ |
| **الركود الانكماشي (Deflation)** | سالب | منخفض/منكمش | السندات الحكومية، الذهب | الأسهم، السلع | سلبي قوي |
| **الانتعاش التضخمي (Reflation)** | قوي/صاعد | صاعد | السلع، الأسهم الدورية | السندات طويلة الأجل | إيجابي متوسط |

### قاعدة التطبيق:
- **الخطوة الأولى دائماً:** حدد النظام الحالي قبل أي تحليل فردي للأصول.
- **التضخم الركودي** هو "القاتل الصامت" لمحفظة 60/40 — السندات لا تحمي لأن التضخم يرفع العوائد بينما الركود يضرب الأسهم.

## 2. القواعد الذهبية للارتباطات (Golden Correlation Rules)

### 2.1 قاعدة الارتباط التربيعي (أسهم ↔ سندات)
الارتباط بين الأسهم والسندات **ليس ثابتاً** — بل يتبع منحنى على شكل U:
- **التضخم قرب الهدف (2%):** ارتباط سلبي → التنويع يعمل.
- **التضخم مرتفع جداً أو منخفض جداً:** ارتباط إيجابي → التنويع يفشل.
- **النموذج:** `ρ = α₀ + α₁(INFL) + α₂(INFL²) + α₃(VIX) + α₄(CONF)`

### 2.2 قاعدة فك ارتباط الذهب والدولار
- **تاريخياً:** ارتباط عكسي (-0.5 إلى -0.8).
- **2023-2026:** انكسر النمط — ارتفعا معاً بسبب **De-dollarization** وشراء البنوك المركزية للذهب.
- **القاعدة:** إذا ارتفع الذهب والدولار معاً → السبب **جيوسياسي/هيكلي** وليس نقدياً → لا تراهن على عودة الارتباط العكسي.

### 2.3 قاعدة القيادة الزمنية (السندات تقود)
- السندات تسعر الركود قبل الأسهم بـ **3-6 أشهر**.
- انعكاس المنحنى (2s10s) → الركود الفعلي يأتي عند **عودة الانحدار (Steepening)**.
- **تسطح المنحنى في 2026:** من 71.5 bps إلى 46.3 bps → إشارة أن التشدد النقدي يكبح النمو.

### 2.4 قاعدة العوائد الحقيقية (Real Yields)
- **الذهب والبيتكوين** يتفاعلان مع العائد الحقيقي (TIPS) وليس الفائدة الاسمية.
- **العتبة الحرجة:** فائدة حقيقية فوق 1.5%-2.0% → رياح معاكسة قوية للذهب وأسهم النمو.
- **الاستثناء:** تنكسر القاعدة في حالات "القمع المالي" أو فقدان الثقة في الديون السيادية.

### 2.5 قاعدة أزمة السيولة (Correlation Collapse)
- في صدمات **Margin Calls** → كل الأصول تهبط معاً → الدولار الكاش هو الملاذ الوحيد.
- **علامات الإنذار:** ارتفاع حاد في VIX + MOVE معاً، اتساع فروق الائتمان (HYG/TLT).
- في هذه الحالة: **أوقف كل الخوارزميات فوراً**.

### 2.6 قاعدة عملات السلع (Commodity FX)
- AUD مرتبط بالنمو الآسيوي/الصيني أكثر من أسعار الطاقة.
- NOK/CAD مرتبطان بالنفط لكن يتأثران بتدفقات الملاذ الآمن.
- **القاعدة:** حلل "المنطقة الجغرافية المهيمنة على الطلب" وليس فقط سعر السلعة.

## 3. ميكانيكا السيولة (Liquidity Mechanics)

### 3.1 التدفق مقابل الرصيد (Flow vs Stock)
- **تدفق** التيسير الكمي (QE) هو ما يدفع الأسعار، وليس حجم الميزانية العمومية.
- QE → يزيد نسبة النقد إلى الأصول → يدفع المستثمرين نحو "البحث عن العائد" (Reach for Yield).
- QT → يسحب النقد → ضغط هبوطي + اتساع فروق الائتمان.

### 3.2 قناة السيولة-المدة في البنوك
- QT يستنزف احتياطيات البنوك (أكثر أصولها سيولة).
- البنوك ترد بـ:
  - **تقليص الإقراض** → تشديد ائتماني حقيقي.
  - **تمديد المدة (Duration Extension)** → مزيد من المخاطر الكامنة.
- **الخطر:** الالتزامات قصيرة الأجل لا تنكمش بسرعة الاحتياطيات → **فجوة سيولة** → حوادث مفاجئة.

### 3.3 السيولة العالمية المتقاطعة
- QT أمريكي قد يُعوَّض جزئياً بتحفيز صيني أو تيسير ياباني.
- **القاعدة:** راقب السيولة العالمية الصافية (Net Global Liquidity) وليس فقط الاحتياطي الفيدرالي.

## 6. قواعد التطبيق الخوارزمي (Algorithmic Implementation Rules)

### الفلتر 1: كشف النظام (Regime Detection)
```
IF yield_curve_2s10s < 0 AND CPI > 3.5% THEN regime = "STAGFLATION"
IF yield_curve_2s10s > 50bps AND CPI < 2.5% THEN regime = "GOLDILOCKS"
IF unemployment_rate_change > +0.5% AND CPI < 2% THEN regime = "RECESSION"
IF GDP_growth > 2.5% AND CPI > 3% THEN regime = "REFLATION"
```

### الفلتر 2: إيقاف الطوارئ (Emergency Kill Switch)
```
IF VIX > 35 AND MOVE > 150 AND HYG/TLT_ratio declining > 3% weekly:
    HALT ALL ALGORITHMS
    REASON: "Correlation Collapse - Liquidity Crisis Detected"
```

### الفلتر 3: التحقق من صحة الارتباط (Correlation Validation)
```
IF Gold_rising AND DXY_rising simultaneously:
    FLAG: "Geopolitical/Structural demand - NOT monetary"
    DO NOT short Gold based on DXY strength

IF real_yield_TIPS > 2.0%:
    REDUCE exposure to: Gold, BTC, Growth Stocks
    INCREASE exposure to: Short-duration bonds, Cash

IF NDX/RUT_ratio rising sharply:
    FLAG: "Fragile Rally - Liquidity concentration"
    REDUCE overall equity beta
```
